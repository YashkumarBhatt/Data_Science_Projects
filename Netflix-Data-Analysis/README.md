# 🍿 Netflix Data Analysis & Visualization

## Overview
This project performs an exploratory data analysis (EDA) on a comprehensive dataset of Netflix movies and TV shows. The analysis uncovers trends over time, top content-producing countries, and popular genres within the platform's vast catalog.

## Key Objectives
1. **Data Cleaning & Processing:** Drop duplicates, handle missing values in director and country columns, and parse date objects for temporal feature engineering.
2. **Exploratory Data Analysis (EDA):** Discover trends over time, compare Movies vs. TV Shows, and identify the most active countries producing content.
3. **Visualization:** Create rich visualizations using `matplotlib`, `seaborn`, and `wordcloud` to showcase content ratings, durations, and genre text analysis.

## Tech Stack
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, WordCloud

## Findings
- **Content Dominance:** Movies make up roughly 70% of the catalog, while TV shows comprise 30%.
- **Global Reach:** The US and India are the largest contributors to the platform.
- **Audience & Content:** The platform leans heavily towards mature audiences (TV-MA), with International Movies, Dramas, and Comedies being the most prominent genres.
